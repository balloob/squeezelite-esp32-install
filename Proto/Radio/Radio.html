Plateform : Muse Proto

Purpose: This implements a simple internet radio project

Features of this Squetch:

  • Streams mp3 radio
  • Wifi hotspot to change radio presets and change wifi credentials
  • Mute button, volume UP, Down

Once installed search for the wifi ssid "muse" password is "musemuse" once connected open a browser to 192.168.4.1 to setup your wifi internet access
